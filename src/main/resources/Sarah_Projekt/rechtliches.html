Data Privacy


I. Preamble

With this privacy policy, www.wecup.nl (hereinafter referred to as "wecup") declares which personal data is processed during a visit to the www.wecup.nl website. Personal data includes all information that can be used to identify a person and that can be traced back to a person - for example, the name, e-mail address, telephone number and IP address. wecup takes data protection very seriously and treats personal data confidentially and in accordance with legal requirements. It should be noted that data transmission on the Internet can always be subject to security vulnerabilities. A full protection against access by third parties is not feasible.


II. Legal Basis

Wecup collects, uses and discloses personal data only if this is permitted by law, is necessary for the contractual purpose or if the user consents to the processing of data. The duration of the storage of the data is limited with the duration of the registration of your user profile. After (anytime) termination of the registration, your personal data will be deleted immediately and completely.


Certain visitor data is also processed via server log files (website visited, time of access, amount of data sent in bytes, source/reference from which you accessed the page, browser used, operating system used, IP address used). The data collected is only used for statistical analysis and to improve the website. All data is processed anonymously, which means that a natural person is no longer identifiable.


III. Person responsible for data processing

The data controller is Wecup BV, Randomstraat 123, 1234 Amsterdam, NL, phone: +(12) 345 67 89, mail: info@wecup.nl. Wecup has appointed a data protection officer. This person can be reached at info@wecup.nl


IV. Rights of Users

You may contact wecup.nl at any time to exercise your lawful rights under the General Data Protection Regulation:


  • Right to information about their stored personal data, its origin and recipients and the purpose of data processing
  • Right to rectification
  • Right to erasure of incorrect or unlawfully processed data
  • Right to object
  • Right to withdraw consent
  • Right to restriction of processing, as well as blocking
  • Right to data portability

Insofar as there are changes to your personal data, we request that you notify us accordingly. The following rights can be implemented independently in the user profile:

  • Right to rectification
  • Right to erasure
  • Right to data portability

These requests for information, deletion, rectification, objection and / or data transfer, in the latter case, provided that this does not cause a disproportionate effort, should be addressed to the Data Protection Officer at dataprivacy@wecup.nl. Please include clear and comprehensible proof of identification so that we can send the data protection-compliant response to the person actually affected. karriere.at will respond to the request within one month at the latest and, in the event of a rejection, will provide reasons for the rejection. In the event of a justified request for deletion or correction, this will also be forwarded to the recipients of personal data for implementation by automated means. You can also exercise your right to information regarding your data stored by karriere.at by viewing your user profile.



Cookies


In order to improve the technical function of the website, analyze user behavior, as well as to be able to submit interesting job offers to our users by our advertising partners in our opinion (remarketing), this website uses so-called "cookies". The use of cookies increases the user-friendliness and security of this website. Cookies are small text files that a website stores on the visitor's computer when they visit and store information about the user and the website. Cookies can serve different purposes. The most common application is the recognition of users. We store this data for a maximum period of 36 months. Common browsers offer the setting option to not allow cookies. Note: It is not guaranteed that you will be able to access all functions of this website without restrictions if you make the appropriate settings.


In order to constantly improve our offerings, we use various analysis tools that record and evaluate the interactions of randomly selected individual visitors with our websites. For example, mouse movements and clicks are recorded with the aim of identifying potential improvements to the respective website. In addition, information on the operating system, browser, incoming and outgoing references (links), geographical origin, as well as resolution and type of device are evaluated for statistical purposes. This information is not passed on to third parties and is only stored for a maximum of one month. The recording of this data can be disabled by setting the DoNotTrack header in the browser.
